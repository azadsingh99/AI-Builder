# mongo-crud-builder

`mongo-crud-builder` is a simple and efficient Node.js library designed to streamline the process of creating CRUD (Create, Read, Update, Delete) operations for MongoDB. It also helps you to create the model of the MongoDB Dynamically, just pass the name and fields of the models from the frontend or the postman it will automatically create the model and will also generate its dynamic CRUD API into the Rest with the NodeJS. This package provides a flexible and easy-to-use API that allows developers to quickly set up CRUD operations without having to write repetitive code.

## Features

- **Ease of Use**: Simplifies the creation of CRUD operations with minimal setup.
- **Flexibility**: Supports dynamic schema definitions, allowing for customizable models.
- **Validation**: Integrates with Mongoose to provide schema validation.
- **Error Handling**: Provides robust error handling for database operations.
- **Extensibility**: Easily extendable to add custom methods and middleware.
